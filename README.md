GoBang
====
一款五子棋app，支持双人模式，wifi，蓝牙联机对战模式

## 主要运用
* mvp模式
* wifi直连库[salut](https://github.com/markrjr/Salut)
* [materialdesign](https://github.com/navasmdc/MaterialDesignLibrary)
* [eventbus](https://github.com/square/otto)
